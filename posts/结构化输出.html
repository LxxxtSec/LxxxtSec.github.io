结构化输出的原理

​ Spring AI 实现结构化输出的思路：

  1. 分析你传入的 Java 类（Record 或 POJO），生成对应的 JSON Schema；
  2. 把 JSON Schema 附加到 Prompt 中，告诉模型“请按照这个格式输出”；
  3. 拿到模型返回的 JSON 后，自动反序列化成 Java 对象。

​ 只需要定义目标属性，调一个方法。
